# Frontend Development Notes

## Current Branch Context

Frontend development should currently be based on the `front_back_merge` branch which contains the unified frontend/backend structure for BudgetWise.

## Frontend Responsibilities

Frontend work includes:

• Implementing UI pages from Figma designs  
• Maintaining reusable UI components  
• Connecting frontend pages to backend API endpoints  
• Ensuring UI consistency across dashboard, expenses, and budgeting views  

## Workflow

1. Pull latest changes from `front_back_merge`
2. Create feature branches for frontend changes
3. Test changes locally
4. Push updates and coordinate with backend integration

## Immediate Priorities

• Dashboard page integration  
• Expense tracking UI  
• Budget / calendar view alignment  
• Frontend/backend API integration
