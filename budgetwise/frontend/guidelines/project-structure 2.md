# BudgetWise Frontend Project Structure

This document describes the current frontend layout after the repository merge.

## Main directories

app/
Next.js application routing and page layouts.

src/
Shared components and frontend logic.

guidelines/
Documentation and development notes for the frontend team.

## Current status

The frontend structure is still being aligned after merging multiple branches.

The main focus moving forward is:

- integrating dashboard UI with backend data
- implementing expense tracking views
- connecting the budgeting calendar
- cleaning up components exported from Figma

## Development notes

Frontend contributors should work from the `front_back_merge` branch and create feature branches for UI work.
